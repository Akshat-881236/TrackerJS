# TrackerJS

TrackerJS is a privacy-safe, client-side visit memory and revisit reminder system
created by **Akshat Prasad**.

## What it does
- Remembers visited projects and pages (locally)
- Helps users continue where they left off
- Shows revisit reminders and recent history
- Improves navigation across related projects

## What it does NOT do
- No cookies
- No analytics
- No server calls
- No user tracking
- No SEO manipulation

## Privacy
All data is stored **only in the user's browser (localStorage)**.
Nothing is sent anywhere.

## Designed to work with
- SitemapJS (SEO infrastructure)
- ProjectJS (project network graph)
